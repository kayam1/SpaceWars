
# Overview

A classic arcade-style spaceship shooter with pixel art graphics. Control your ship, defeat waves of enemies, and survive as long as possible.

## Gameplay

### Objective

* Destroy enemy ships to earn points

* Survive as long as possible

* Achieve the highest score


### Player

* Health: 100 HP

* Movement: follows the mouse cursor

* Firing: Continuous laser shots

* Invincibility frames after taking damage

### Enemies

* Spawn from the top of screen

* Move downward

* Shoot lasers at the player

* Enemies get stronger with each passing wave

## Key Features

  🎮 Responsive controls - Smooth ship movement
  
  💥 Combat system - Lasers, health, damage
  
  📊 Health bar - Visual player health indicator
  
  🎨 Pixel art - Retro-style graphics
  
  🔊 Sound effects - Lasers, explosions, impacts
